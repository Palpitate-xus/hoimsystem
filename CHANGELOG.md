# 更新日志

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范。

## [1.1.0] - 2026-05-18

### 新增

- **住院全流程**：病区床位管理、入院登记、住院医嘱（长期/临时）、护士工作站、住院费用、出院结算
- **手术麻醉管理**：手术申请、手术排台、麻醉记录
- **电子病历进阶**：结构化病历、病程记录、查房记录、病历模板、CA 数字签名
- **体检中心**：体检套餐、项目管理、体检预约、记录录入、报告生成
- **多学科会诊（MDT）**：会诊申请、参与科室管理、结果录入
- **临床路径**：路径模板、患者路径跟踪
- **双向转诊**：门诊/住院/急诊转诊申请、处理流程
- **预交金账户**：充值、查询、扣费、退款
- **耗材管理**：医疗耗材入库、查询、修改、删除
- **药品采购**：采购订单创建、审批、入库
- **ADR 监测**：药品不良反应上报、跟踪、处理
- **不良事件上报**：医疗安全事件记录
- **分诊台管理**：4 级分诊、科室分流
- **候诊巡视**：巡视记录、关注高风险患者
- **复诊随访**：随访计划、患者反馈
- **操作日志自动记录**：FastAPI 中间件拦截 POST/PUT/DELETE 操作，自动写入审计日志

### 优化

- **操作日志中间件**：从 accesstoken 解析用户、路径自动映射为中文操作+对象
- **响应时间脱敏**：StripMicrosecondMiddleware 自动去除 datetime 微秒部分
- **身份证号脱敏**：前端 maskIdentity 函数统一脱敏显示（370101********1234）
- **三步式业务流程**：现场挂号、报到签到、双向转诊改为三步式（输入身份证→选项卡片→成功）
- **UI 全面美化**：
  - 移除 30+ 页面表格中的 ID/UUID 列
  - 所有 el-table 添加 empty-text 空状态
  - 所有列表加 v-loading 加载状态
  - 表单手动输入 ID 改为 el-select 下拉
  - 状态列统一使用 el-tag 美化
  - 搜索框 placeholder 改为具体提示
- **文字表述优化**：移除模板自带的"vue-admin-better"等字样，统一为医疗系统术语
- **首页快捷入口**：根据用户角色动态显示常用功能入口
- **通知公告点击跳转**：首页公告时间线可点击查看详情

### 修复

- 修复 main.py 中重复创建 FastAPI 实例的 bug
- 修复前端挂载点 ID 不一致导致的空白页（index.html ↔ main.js）
- 修复 vab.scss 注释格式语法错误
- 修复 admin/doctor1 测试账号缺失关联数据
- 修复 MDT 会诊显示科室 ID 而非名称
- 修复 birthday/exam_date 等日期字段错误显示时分秒
- 修复 ExamResult.doctor 关联导致的 SQLAlchemy 错误
- 修复多个页面 API 导入名错误（getScheduleList、getAppointmentsForCheckin）
- 过滤 rspack dev server 的 ResizeObserver 无害运行时错误

### 文档

- 新增 `doc/README.md` 文档导航中心
- 新增 `doc/architecture.md` 系统架构文档
- 新增 `doc/user-manual.md` 用户操作手册（650+ 行，覆盖所有角色）

## [1.0.0] - 2026-05-10

### 新增

- **门诊全流程管理**：覆盖挂号、就诊、收费、取药、检验、随访等门诊完整流程
- **多角色支持**：管理员、科室主任、医生、护士、收费员、患者、导医/挂号员
- **预约挂号**：在线预约医生号源，支持现场挂号和窗口挂号
- **排队叫号**：候诊队列管理、叫号、过号、跳号、急诊优先
- **电子病历**：病历书写、查询、修改，关联挂号/预约记录
- **处方管理**：医生开处方、库存校验、处方审核、发药确认、退药处理
- **收费管理**：费用计算、在线缴费、退费、发票管理、日结对账
- **检查检验**：检查申请、结果录入、危急值报告、样本全流程跟踪
- **药品管理**：药品入库、查询、库存预警、库存盘点、处方点评
- **智能导诊**：310+ 症状关键词匹配，推荐科室和医生
- **报表统计**：门诊量、财务、药品消耗、医生工作量多维统计
- **消息中心**：站内信/短信通知、消息列表、已读标记
- **支付接口**：微信/支付宝 Mock 支付、现金支付、支付流水
- **权限分配**：用户角色管理、密码重置、用户删除
- **医生考勤**：每日签到/签退、出勤统计
- **数据备份恢复**：手动备份、恢复、下载、删除
- **操作日志审计**：全量操作记录、筛选查询
- **系统管理**：数据字典、系统参数配置
- **前端页面美观性**：统一搜索栏、分页、表单样式，响应式布局

### 安全

- 密码加密存储（bcrypt）
- JWT Token 认证替代纯字符串 Token
- 密码传输 RSA 加密
- 接口角色权限校验细化
- 敏感配置项改为环境变量注入

### 基础设施

- Docker 支持（Dockerfile + docker-compose.yml）
- GitHub Actions CI/CD 工作流
- 部署文档（deployDoc.md）
- 贡献指南（CONTRIBUTING.md）
- 开源许可证（MIT）
