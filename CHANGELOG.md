# 更新日志

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范。

## [1.0.0] - 2026-05-10

### 新增

- **门诊全流程管理**：覆盖挂号、就诊、收费、取药、检验、随访等门诊完整流程
- **多角色支持**：管理员、科室主任、医生、护士、收费员、患者、导医/挂号员
- **预约挂号**：在线预约医生号源，支持现场挂号和窗口挂号
- **排队叫号**：候诊队列管理、叫号、过号、跳号、急诊优先
- **电子病历**：病历书写、查询、修改，关联挂号/预约记录
- **处方管理**：医生开处方、库存校验、处方审核、发药确认、退药处理
- **收费管理**：费用计算、在线缴费、退费、发票管理、日结对账
- **检查检验**：检查申请、结果录入、危急值报告、样本全流程跟踪
- **药品管理**：药品入库、查询、库存预警、库存盘点、处方点评
- **智能导诊**：310+ 症状关键词匹配，推荐科室和医生
- **报表统计**：门诊量、财务、药品消耗、医生工作量多维统计
- **消息中心**：站内信/短信通知、消息列表、已读标记
- **支付接口**：微信/支付宝 Mock 支付、现金支付、支付流水
- **权限分配**：用户角色管理、密码重置、用户删除
- **医生考勤**：每日签到/签退、出勤统计
- **数据备份恢复**：手动备份、恢复、下载、删除
- **操作日志审计**：全量操作记录、筛选查询
- **系统管理**：数据字典、系统参数配置
- **前端页面美观性**：统一搜索栏、分页、表单样式，响应式布局

### 安全

- 密码加密存储（bcrypt）
- JWT Token 认证替代纯字符串 Token
- 密码传输 RSA 加密
- 接口角色权限校验细化
- 敏感配置项改为环境变量注入

### 基础设施

- Docker 支持（Dockerfile + docker-compose.yml）
- GitHub Actions CI/CD 工作流
- 部署文档（deployDoc.md）
- 贡献指南（CONTRIBUTING.md）
- 开源许可证（MIT）
