# 业务数据字典

> 本文档汇总系统中所有枚举值、状态码、业务字典的统一定义。
> 用于开发对照、测试用例编写、数据分析时参考。
> 数据库表字段详细定义见 [databaseDoc.md](databaseDoc.md)。

---

## 一、用户角色

| 编码 | 中文 | 说明 |
|:----:|:----:|:----|
| `admin` | 管理员 | 全局管理：用户、医生、病人、科室、药品、通知 |
| `director` | 科室主任 | 本科室管理：排班、医生、处方点评 |
| `doctor` | 医生 | 诊疗：病历、处方、检查申请、考勤 |
| `nurse` | 护士 | 护理：分诊、生命体征、住院护理 |
| `cashier` | 收费员 | 收费：窗口挂号/退号、费用结算 |
| `pharmacist` | 药师 | 药房：处方审核、发药、库存 |
| `guide` | 导诊员 | 导诊：分诊台、智能导诊、候诊队列 |
| `patient` | 患者 | 自助：挂号/预约、缴费、查报告 |

## 七、系统配置项

部分配置项通过 `hoimsystem_config` 表维护：

| 配置键 | 默认值 | 说明 |
|:------|:------:|:----|
| `hospital_name` | 示范医院 | 医院名称 |
| `appointment_limit` | 3 | 每人每日预约上限 |
| `registration_fee` | 10 | 挂号费（元） |
| `breach_threshold` | 3 | 违约阈值（次） |
| `breach_suspend_days` | 30 | 违约暂停天数 |
| `prepaid_warning_amount` | 500 | 预交金告警金额 |
| `low_stock_threshold` | 10 | 低库存告警阈值 |
| `near_expiry_days` | 30 | 近效期告警天数 |

> 通过 [system/config](../vue3-new-ui/src/views/system/config.vue) 页面管理。

## 八、字段命名约定参考

| 字段类型 | 命名 | 示例 |
|:------:|:----|:----|
| 主键 | `{表名简称}_id` | `patient_id`, `prescription_id` |
| 外键 | 与目标表主键同名 | `patient_id` |
| 时间 | `{动词}_time` | `create_time`, `audit_time`, `discharge_time` |
| 状态值 | `status` | `status = 0/1/2/3` |
| 状态文本（仅响应） | `status_text` | `"未就诊"` |
| 标志位 | `is_xxx` | `is_active`, `is_deleted` |
| 数量 | `xxx_count` 或 `xxx_num` | `bed_count` |
| 金额（元） | `xxx_amount` | `total_amount` |

## 九、维护约定

新增字典项时：

1. 在本文档对应章节添加
2. 数据库表注释同步更新（`comment="..."`）
3. 后端 API 返回时附带 `xxx_text` 字段
4. 前端组件使用一致的中文映射

发现不一致或缺失？提 PR 完善！
