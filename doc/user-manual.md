## 一、系统概述

### 1.1 系统简介

HOIM（Hospital Outpatient Information Management）医院门诊信息管理系统，是一款面向中小型医院的综合信息管理平台。系统覆盖门诊、住院、药房、收费、检验、体检等核心业务，支持管理员、医生、护士、收费员、患者等多角色协同工作。

### 1.2 系统角色与权限

| 角色 | 权限说明 | 主要功能 |
|------|----------|----------|
| **管理员(admin)** | 全部权限 | 用户管理、科室管理、药品管理、系统配置、数据备份 |
| **科室主任(director)** | 管理+医生权限 | 医生工作站、药房管理、报表统计、住院管理 |
| **医生(doctor)** | 诊疗权限 | 写病历、开处方、开检查、查看排班 |
| **患者(patient)** | 自助服务权限 | 预约挂号、缴费、查报告、健康档案 |

### 1.3 界面布局

登录后系统左侧为导航菜单，顶部为面包屑导航和用户信息区，中间为主内容区。

- **左侧菜单**：根据当前用户角色动态显示可用功能模块
- **顶部工具栏**：显示当前页面标题、用户头像、消息通知
- **主内容区**：显示当前功能页面的操作内容

## 二、登录与注册

### 2.1 登录系统

1. 打开浏览器，访问系统地址（如 `http://localhost:8091`）
2. 进入登录页面，输入**账号**和**密码**
3. 点击"登录"按钮

**测试账号：**

| 角色 | 账号 | 密码 |
|------|------|------|
| 管理员 | `admin` | `admin123` |
| 科室主任 | `director1` | `director123` |
| 医生 | `doctor1` | `doctor123` |
| 患者 | `patient1` | `123456` |

### 2.2 患者注册

1. 在登录页面点击"注册账号"
2. 填写注册信息：
   - **用户名**：建议使用身份证号
   - **密码**：至少6位
   - **身份证号**：18位有效身份证号码
   - **手机号**：11位手机号码
   - **性别**：男/女
   - **地址**：居住地址
   - **出生日期**：选择日期
3. 点击"注册"完成

> 提示：注册成功后自动登录，账号角色默认为"患者"。

### 2.3 退出登录

点击页面右上角用户头像，选择"退出登录"即可安全退出系统。

## 三、管理员操作指南

### 3.1 医生管理

**路径**：管理员模块 > 医生管理

#### 新增医生
1. 点击"新增医生"按钮
2. 填写医生信息：
   - **用户名**：医生登录账号
   - **密码**：初始密码
   - **姓名**：医生真实姓名
   - **职称**：主任医师/副主任医师/主治医师/住院医师
   - **性别**：男/女
   - **手机号**：11位号码
   - **所属科室**：下拉选择
   - **权限**：doctor（普通医生）/ director（科室主任）
   - **学历**：本科/硕士/博士
3. 点击"确定"保存

#### 编辑/删除医生
- 在列表中找到目标医生，点击"编辑"修改信息
- 点击"删除"可移除医生账号（谨慎操作）

### 3.2 科室管理

**路径**：管理员模块 > 科室管理

1. 点击"新增科室"，填写科室名称、联系电话、科室位置
2. 科室主任可先留空，后续在医生管理中指定
3. 编辑和删除操作与医生管理类似

### 3.3 患者管理

**路径**：管理员模块 > 病人管理

- 查看所有注册患者的基本信息
- 支持按姓名、身份证号搜索
- 可编辑患者信息（姓名、性别、手机号、地址、过敏史等）

### 3.4 通知公告

**路径**：管理员模块 > 通知公告

1. 点击"发布公告"，填写标题和内容
2. 勾选"紧急通知"可使公告标记为红色
3. 选择"发送对象"：全部用户 / 仅医生 / 仅患者
4. 设置"过期时间"，超过该时间公告自动失效
5. 首页右侧"系统公告"区域会展示最新5条公告

> 提示：点击首页公告标题可查看完整内容。

### 3.5 号源池管理

**路径**：管理员模块 > 号源池管理

用于管理各科室医生的排班和号源数量：
1. 选择科室和医生
2. 设置日期、时段（上午/下午）
3. 设置号源数量（该时段可预约人数）
4. 标记是否为"专家号"

## 四、患者操作指南

### 4.1 智能导诊

**路径**：患者服务 > 智能导诊

1. 选择或输入症状描述
2. 系统根据症状推荐合适的科室和医生
3. 点击推荐结果可直接跳转到预约挂号页面

### 4.2 预约挂号

**路径**：患者服务 > 预约挂号

#### 预约流程
1. 选择科室（如"内科"、"外科"）
2. 选择日期（只能预约未来7天内的号源）
3. 系统显示该科室当天可预约的医生列表
4. 选择医生和时段（上午/下午）
5. 确认预约信息，点击"提交预约"

> 提示：蓝色卡片表示普通号，红色卡片表示专家号。

#### 查看/取消预约
- 在预约列表中可查看所有预约记录
- 状态说明：
  - **待就诊**：尚未到预约日期
  - **已就诊**：已完成就诊
  - **已取消**：主动取消的预约
- 点击"取消"可取消未过期的预约

### 4.3 现场挂号

**路径**：患者服务 > 现场挂号

适用于已到院但未预约的患者：
1. 选择科室
2. 选择医生（显示当前有号的医生）
3. 确认挂号，系统自动分配排队序号
4. 挂号成功后前往候诊区等待叫号

### 4.4 缴费管理

**路径**：患者服务 > 缴费管理

1. 系统自动列出待缴费项目（医生开具的处方和检查单）
2. 点击"去支付"，选择支付方式：
   - **微信支付**：扫码支付
   - **支付宝支付**：扫码支付
   - **现金支付**：到收费窗口缴费
3. 支付成功后状态变为"已缴费"
4. 可查看历史缴费记录

> 提示：已缴费项目不可重复支付。

### 4.5 病历查询

**路径**：患者服务 > 病历查询

- 查看历次就诊的病历记录
- 包含主诉、诊断结果、开方医生等信息
- 支持按日期筛选

### 4.6 处方查询

**路径**：患者服务 > 处方查询

- 查看所有处方明细（药品名称、数量、用法）
- 处方状态：待审核 / 已审核 / 已发药

### 4.7 健康档案

**路径**：患者服务 > 健康档案

- 查看个人基本信息
- 查看过敏史（如有）
- 查看预交金余额（住院患者）

### 4.8 就诊评价

**路径**：患者服务 > 就诊评价

1. 选择已就诊的医生
2. 打分（1-5星）
3. 填写评价内容（可选）
4. 提交评价

## 五、医生操作指南

### 5.1 查看排班

**路径**：医生工作站 > 医生排班

- 查看个人周排班表
- 查看各时段号源余量
- 科室主任可调整排班

### 5.2 写病历

**路径**：医生工作站 > 病历管理

#### 新建病历
1. 选择患者（支持搜索患者姓名或身份证号）
2. 填写病历内容：
   - **主诉**：患者自述的主要症状
   - **现病史**：症状发生时间、经过
   - **既往史**：过往疾病、手术史
   - **诊断**：初步诊断结果
3. 点击"保存"或"提交"

#### 使用病历模板
1. 点击"模板"按钮
2. 选择常用病历模板（如"上呼吸道感染模板"）
3. 系统自动填充模板内容，医生按需修改
4. 保存为个人常用模板

### 5.3 开处方

**路径**：医生工作站 > 处方管理

#### 新建处方
1. 选择患者
2. 点击"添加药品"
3. 搜索药品名称，选择药品
4. 设置：数量、用法用量、频次（如 qd-每日一次、bid-每日两次）
5. 点击"提交处方"

> 提示：系统会自动检查药品库存和过敏史冲突，如有问题会弹窗提示。

#### 处方审核状态
- **待审核**：提交后等待药师审核
- **已审核**：药师审核通过
- **已发药**：药房已完成发药

### 5.4 开检查/检验单

**路径**：医生工作站 > 检查检验申请

1. 选择患者
2. 选择检查类型（血常规、尿常规、B超、CT、MRI等）
3. 勾选具体检查项目
4. 标记是否加急
5. 提交申请

### 5.5 考勤签到

**路径**：医生工作站 > 考勤签到

1. 上班时点击"签到"按钮
2. 系统自动记录签到时间
3. 9:00 后签到标记为"迟到"
4. 下班时点击"签退"

### 5.6 多学科会诊（MDT）

**路径**：医生工作站 > 多学科会诊

#### 发起会诊
1. 点击"新增会诊"
2. 输入患者ID
3. 填写初步诊断
4. **选择参与科室**（多选下拉框，如内科、外科、影像科）
5. 提交申请

#### 查看会诊
- 列表显示所有会诊记录
- 参与科室以标签形式展示（如"内科 外科 影像科"）
- 状态：待会诊 / 会诊中 / 已完成
- 会诊完成后可录入会诊结果

### 5.7 临床路径

**路径**：医生工作站 > 临床路径

- 查看各病种的标准诊疗路径
- 患者入径/出径管理
- 路径执行进度跟踪

## 六、药房操作指南

### 6.1 处方审核与发药

**路径**：药房管理 > 处方审核与发药

#### 审核处方
1. 查看待审核处方列表
2. 点击处方查看药品明细
3. 检查：
   - 药品剂量是否合理
   - 是否存在配伍禁忌
   - 患者是否有过敏史
4. 点击"审核通过"或"审核不通过"

#### 发药
1. 在已审核列表中找到处方
2. 核对药品和数量
3. 点击"确认发药"
4. 系统自动扣减库存

### 6.2 药品管理

**路径**：药房管理 > 药品管理

- 查看药品目录和库存
- 新增药品：填写名称、规格、库存、价格、有效期、供应商
- 编辑药品信息
- 删除药品（谨慎操作）

### 6.3 库存预警

**路径**：药房管理 > 库存预警

- 自动显示低库存药品（库存低于预警线）
- 显示即将过期药品（30天内过期）
- 点击可跳转到采购申请

### 6.4 处方点评

**路径**：药房管理 > 处方点评

- 查看已发药处方
- 对处方合理性进行评价打分
- 记录点评意见

## 七、收费员操作指南

### 7.1 窗口挂号

**路径**：收费管理 > 窗口挂号

为未预约患者办理现场挂号：
1. 输入患者身份证号或选择已有患者
2. 选择科室和医生
3. 收取挂号费
4. 打印挂号单

### 7.2 费用管理

**路径**：收费管理 > 费用管理

- 查看所有待收费项目
- 显示患者姓名、开方医生、金额、状态
- 点击"收费"完成收款
- 支持退费操作（需填写退费原因）

### 7.3 发票管理

**路径**：收费管理 > 发票管理

1. 选择已缴费记录
2. 点击"开具发票"
3. 系统自动生成发票号码
4. 支持打印发票

### 7.4 日结对账

**路径**：收费管理 > 日结对账

1. 选择对账日期
2. 系统自动汇总当日：
   - 收费总金额
   - 退费总金额
   - 各支付方式金额（微信/支付宝/现金）
   - 收费笔数
3. 核对无误后确认日结

## 八、护士操作指南

### 8.1 生命体征录入

**路径**：护士预检 > 生命体征录入

1. 选择或输入患者
2. 录入体征数据：
   - 体温（℃）
   - 血压（收缩压/舒张压 mmHg）
   - 脉搏（次/分）
   - 体重（kg）
3. 点击"保存"

### 8.2 候诊巡视

**路径**：排队叫号 > 候诊巡视

1. 点击"新增巡视记录"
2. 选择患者
3. 填写巡视内容（如"患者自述胸闷，已通知医生"）
4. 选择状态：正常 / 需关注 / 急诊绿色通道
5. 提交记录

## 九、检验科操作指南

### 9.1 样本接收

**路径**：检验科 > 检查结果录入

1. 在"待接收"标签页查看待处理申请
2. 扫描或输入样本编号
3. 确认样本信息（患者、检查项目）
4. 点击"接收样本"

### 9.2 结果录入

1. 在"待录入"标签页选择检查单
2. 逐项录入检查结果
3. 系统自动比对参考范围，标记异常值
4. 点击"保存"

### 9.3 结果审核

1. 在"待审核"标签页查看已录入结果
2. 核对结果合理性
3. 点击"审核通过"发布报告
4. 患者可在"病历查询 > 检查报告"中查看

## 十、住院管理操作指南

### 10.1 病区床位管理

**路径**：住院管理 > 病区床位

- 查看各病区床位分布图
- 绿色 = 空闲，红色 =  occupied
- 点击床位可查看患者信息

### 10.2 入院登记

**路径**：住院管理 > 入院登记

1. 选择患者（输入身份证号）
2. 选择主管医生和科室
3. 分配病区床位
4. 填写入院诊断、主诉
5. 缴纳押金（可选）
6. 提交完成入院

### 10.3 住院医嘱

**路径**：住院管理 > 住院医嘱

#### 开立医嘱
1. 选择住院患者
2. 选择医嘱类型：长期医嘱 / 临时医嘱
3. 选择类别：药品 / 护理 / 检查 / 治疗
4. 添加医嘱明细（药品名称、剂量、频次、用法）
5. 设置优先级：普通 / 紧急 / 立即
6. 提交医嘱

#### 医嘱执行
- 护士在"护士工作站"中查看待执行医嘱
- 点击"执行"记录执行时间和执行人
- 已执行医嘱不可撤销

### 10.4 护士工作站

**路径**：住院管理 > 护士工作站

- **护理记录**：记录患者生命体征、护理措施
- **体温单**：绘制体温曲线图
- **医嘱执行**：核对并执行医生医嘱

### 10.5 住院费用

**路径**：住院管理 > 住院费用

- 查看患者一日费用清单
- 录入新增费用项目
- 查看费用汇总

### 10.6 出院结算

**路径**：住院管理 > 出院结算

1. 选择待出院患者
2. 系统汇总住院期间全部费用
3. 结算费用（押金多退少补）
4. 填写出院诊断和医嘱
5. 完成出院，床位释放

## 十一、体检中心操作指南

### 11.1 体检套餐管理

**路径**：体检管理 > 体检管理（套餐标签）

1. 新增套餐：填写套餐名称、类别、价格、包含项目
2. 项目用逗号分隔（如"血常规,尿常规,肝功能"）
3. 编辑和删除已有套餐

### 11.2 体检项目管理

**路径**：体检管理 > 体检管理（项目标签）

- 管理体检检查项目库
- 每个项目包含：名称、类别、单位、参考范围、价格

### 11.3 体检预约

**路径**：体检管理 > 体检管理（预约标签）

1. 选择患者
2. 选择体检套餐
3. 选择体检日期
4. 提交预约

### 11.4 体检记录与结果录入

**路径**：体检管理 > 体检管理（记录标签）

1. 选择已报到的体检预约，创建体检记录
2. 在各科室录入分项检查结果
3. 所有项目完成后，医生填写总检结论和建议
4. 标记"完成"，生成体检报告

### 11.5 体检报告

- 在记录列表中点击"查看报告"
- 显示所有检查项目结果和总检结论
- 支持打印报告

### 12.1 数据字典

- 预约限制（每人最多同时预约数）
- 违约阈值（几次违约后暂停预约）
- 库存预警天数

**路径**：系统管理 > 操作日志

## 附录：常见问题

### Q1: 忘记密码怎么办？
请联系管理员重置密码。管理员可在"系统管理 > 权限分配"中重置用户密码。

### Q2: 预约后如何报到？
在"报到签到"页面输入身份证号，系统会自动显示今日待报到预约，选择后点击"确认报到"即可。

### Q3: 为什么看不到某些菜单？
系统根据您的角色自动过滤菜单。如需更多权限，请联系管理员调整。

### Q4: 处方被退回了怎么办？
在"处方管理"中查看退回原因，修改后重新提交。

### Q5: 如何修改个人信息？
点击页面右上角头像，选择"个人中心"即可修改。
